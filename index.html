    More Details - Trojan Automation

    Employs a C# Windows Form Application GUI in order to store data and use that data in a selenium harvestor which stores cookies for request sessions.


    After harvesting a login, a request session can be created in order to provide info about items such as: COVID Compliance, Housing Status, and potential account holds.


    While these features are useful, the main purpose of this project is to complete the trojan check, which is done utilizing requests and the Twilio/Imgur API to send the trojan check to the user's phone.
